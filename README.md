                                                                    Customer Churn Analysis
Project Overview
This project is a comprehensive analysis of customer churn using Power BI. The objective is to identify patterns and factors contributing to customer churn, enabling better decision-making and retention strategies.

Purpose
Understanding why customers leave a service is crucial for any business. This analysis helps in identifying the key drivers of churn and provides insights to improve customer retention through visual storytelling with Power BI.

Data and Features
The dataset includes various customer-related features such as:

Customer Demographics: Age, Gender, Location, etc.
Subscription Details: Plan type, contract type, monthly charges, total charges, etc.
Service Usage: Internet usage, phone services, additional features.
Churn Label: Indicates whether the customer has churned or not.
Approach
Data Cleaning and Preparation:

Handled missing values and prepared the data for analysis.
Created calculated columns for metrics like tenure and monthly charges.
Exploratory Data Analysis:

Performed EDA in Power BI to discover trends, correlations, and patterns.
Visualized data through dashboards including KPIs, charts, and maps for comprehensive insights.
Dashboard Creation:

Built interactive dashboards to explore customer demographics, service usage, and churn rates.
Included filters for deeper insights based on factors like age, service type, and monthly charges.
Key Insights:

Identified features highly correlated with churn.
Highlighted customer segments with high churn probability, allowing for targeted marketing and retention strategies.
Tools
Power BI: Data visualization and dashboard creation
